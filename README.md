# GlowFit Women's Gym

A modern, responsive website for GlowFit Women's Gym built with Next.js 16, React 19, and Tailwind CSS. This website empowers women to achieve their strongest, healthiest selves through personalized fitness programs.

## Features

- **Modern UI/UX Design** - Clean, attractive interface with smooth animations
- **Fully Responsive** - Works on all devices from mobile to desktop
- **Performance Optimized** - Built with Next.js 16 for optimal performance
- **Smooth Scrolling** - Enhanced user experience with smooth scrolling
- **Interactive Components** - Animated components using Framer Motion
- **SEO Optimized** - Proper metadata and structured data for search engines
- **Mobile Navigation** - Responsive hamburger menu for mobile devices

## Sections

1. **Hero Section** - Eye-catching introduction with animated text
2. **Services Section** - Showcase of specialized women's fitness programs
3. **About Section** - Information about the gym's mission and values
4. **Contact Section** - Ways to get in touch with the gym
5. **Footer** - Additional navigation and contact information

## Technologies Used

- [Next.js 16](https://nextjs.org/) - React framework with App Router
- [React 19](https://reactjs.org/) - JavaScript library for building user interfaces
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Framer Motion](https://www.framer.com/motion/) - Animation library for React
- [GSAP](https://greensock.com/gsap/) - Professional-grade animation library
- [React Icons](https://react-icons.github.io/react-icons/) - Popular icons library
- [Lenis](https://github.com/studio-freight/lenis) - Smooth scrolling library

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
